# Todo Application Automation Framework

## 📌 Project Overview

This project is a UI Test Automation Framework built using Selenium WebDriver, Java, TestNG, and Maven. It automates the core functionalities of the Todo Application while following the Page Object Model (POM) design pattern. The framework also integrates Rest Assured for API testing, Allure Reports for reporting, and GitHub Actions for Continuous Integration.

## 🚀 Features

- UI Test Automation using Selenium WebDriver
- API Testing using Rest Assured
- Page Object Model (POM) Design Pattern
- TestNG Test Framework
- Maven Project Management
- Allure Reporting
- WebDriverManager Integration
- Cross-browser support (Chrome & Firefox)
- Headless Chrome execution for CI
- GitHub Actions Continuous Integration
